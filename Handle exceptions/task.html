Description

The author failed to anticipate the potential for user errors such as typos, which resulted in numerous uncaught exceptions and a lack of explicit feedback.

Let's help users of this program with this problem. Add more meaningful feedback for command corner cases. As for the exceptions, find them all and leave an appropriate message.

The list below contains the feedback for command corner cases.
